## Introduction

This is a display of documents prepared for the sample project involving the deployment of the BookStack open-source documentation management system in a corporate infrastructure.

BookStack provides features for organization of user documents into a hierarchical structure of shelves, books, chapters, and pages. Among the main features are a complex search engine, a commenting system and a customizable document access model.

The business documents cover the project brief and description, work schedule, and the results form. The technical documents include the architecture with the complete description of the system configuration and deployment as well as the support procedures definition. The user / admin guide is also provided.
